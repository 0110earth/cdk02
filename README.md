# 概要

本テンプレートは、CDK初回作成用テンプレートとなる。  
基本的にはBLEAv3（https://github.com/aws-samples/baseline-environment-on-aws/blob/main/README_ja.md）に従う。

# 使用方法

1. 本テンプレートをクローンする。
1. `npm install`コマンドを打つ。
1. 「Example」 or 「example」と記載された箇所を適切な名称に変更する。
